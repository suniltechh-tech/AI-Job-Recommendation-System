# AI Job Recommendation System

An AI-powered Job Recommendation System developed using Machine Learning, Flask, and NLP techniques to recommend jobs based on user skills and resumes.

## Features

- Resume Upload System
- Skill-Based Job Recommendation
- TF-IDF Vectorization
- Cosine Similarity Matching
- Match Percentage
- Modern Dashboard UI
- Dark Mode
- Profile System
- Job Search Interface

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Scikit-learn
- Pandas
- NLP
- TF-IDF
- Cosine Similarity

## Project Modules

- Home Page
- Dashboard
- Find Jobs
- User Profile
- Resume Upload
- Recommendation Engine

## Machine Learning Technique

This project uses:
- TF-IDF Vectorization
- Cosine Similarity Algorithm

for intelligent job recommendations.
